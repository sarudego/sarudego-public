A telegram bot

PUBLISHED ON DEC 12, 2019 — TELEGRAM

As a user of the Zaragoza bus service, I've used several applications to get bus times, but fed up with the failures or malfunction, along with my desire to make bots for telegram, I decided to make my own bot to obtain the data of the buses using the Public API of Zaragoza.

This is how I made my first bot.

First the language and the basic objectives:

  • Python 3.7
  • Receive bus times based on a given post number.

Given the multitude of frameworks that there are to create bots with this language, I finally decided on telepot, since it supports asyncio.

I wanted to use poetry as a dependency manager for the application, docker to isolate the application of the system and be able to deploy it in a simple.

Once the base is generated and its operation checked, the Dockerfile is created and it deploys.

So here the first version.

I hope it was useful to discover some tools like poetry or the cookiecutter!

A greeting!
